Help & FAQ

What is this?

The LinkML-Scala playground is an in-browser tool for working with LinkML schemas. Paste a LinkML schema (YAML) and instantly generate JSON Schema, SHACL, RDFS, a Frictionless Table Schema, or Scala classes – or lint and derive/prune your schema. It is powered by LinkML-Scala.

How does it work?

LinkML-Scala is written in Scala 3 and compiled to JavaScript with Scala.js, so the whole generator runs in your browser – there is no backend. The editors are powered by CodeMirror. See the contributing guide for how it is built.

Where is my data processed?

Entirely on your device. Your schema is never uploaded – all generation happens locally in this browser tab. There is no server, no account, and no analytics.

How can I use LinkML-Scala?

Beyond this playground, LinkML-Scala ships three ways:

  • Command-line tool – grab it via the install script, mise, or a prebuilt binary. No Java or Python needed, this is a native binary.
  • JVM / Scala library – on Maven Central.
  • JavaScript / TypeScript library – on npm as @neverblink/linkml.
